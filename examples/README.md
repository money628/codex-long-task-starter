# 示例项目

这里放置 Codex 长任务启动器 / Codex Long Task Starter 生成结果的可读示例。

## simple-todo-app

一个简单 todo 应用示例，适合理解最小 Web app 如何被整理成：

- `project-spec.json`
- `Prompt.md`
- `Plan.md`
- `Implement.md`
- `Documentation.md`
- `Continuity.md`
- `AGENTS.md`
- `START.md`

## chrome-extension

一个 Chrome extension 示例，适合观察带浏览器权限、内容脚本、popup 页面等约束的项目如何进入长任务文件包。

## 使用方式

可以直接打开任一示例目录阅读，也可以把 7 个 Markdown 文件复制到 Codex / OpenCode 中试跑。

示例文件不得包含真实 API Key、Token、账号凭证或私有项目数据。
