# Simple Todo App 项目总需求

## 项目背景

一个适合新手练习的待办事项 Web 应用，支持任务创建、完成、过滤和本地持久化。
## 项目目标

用最小技术栈实现一个清晰、可测试、可扩展的本地 Todo 应用。
## 目标用户

- 学习前端开发的新手
- 需要轻量任务清单的个人用户
## 用户痛点

- 纸质清单不方便修改
- 复杂任务工具学习成本高
- 新手需要一个结构清楚的练习项目
## MVP 范围

- 任务新增
- 任务完成/取消完成
- 任务删除
- 全部/未完成/已完成过滤
- localStorage 持久化
- 空状态和错误提示
## 非目标

- 用户登录
- 云同步
- 团队协作
- 移动端原生应用
- 付费功能
## 核心功能

- 任务输入框
- 任务列表
- 状态过滤
- 本地保存
- 基础响应式布局
## 技术约束

- 任务文本不能为空
- 刷新页面后任务不能丢失
- 界面必须中文化
- 不需要后端
## 禁止事项

- 不要引入数据库
- 不要加入登录系统
- 不要把用户数据上传到服务器
## 交付物

- Prompt.md
- Plan.md
- Implement.md
- Documentation.md
- Continuity.md
- AGENTS.md
- START.md
## Done When 验收标准

- 所有 MVP 功能可用
- 无控制台错误
- 构建通过
- README 说明如何启动
