# Simple Todo App 跨会话状态账本

## 目标

用最小技术栈实现一个清晰、可测试、可扩展的本地 Todo 应用。
## 成功验收标准

- 可以新增任务
- 可以标记完成
- 可以删除任务
- 可以过滤任务
- 刷新后数据保留
- pnpm build 通过
## 约束

- 任务文本不能为空
- 刷新页面后任务不能丢失
- 界面必须中文化
- 不需要后端
## 关键决策

- 用户使用现代浏览器
- 任务量不会超过几百条
- 先用纯本地状态实现，再抽出 storage helper
- 为过滤和持久化写单元测试
