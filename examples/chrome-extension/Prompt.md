# Tech Explain Translator 项目总需求

## 项目背景

选中网页里的英文技术段落后，在侧边栏生成中文解释、术语拆解和可复制摘要。
## 项目目标

让用户在不离开当前网页的情况下理解英文技术内容，并沉淀可复制的中文摘要。
## 目标用户

- 中文开发者
- 技术学习者
- 阅读英文文档的产品工程师
## 用户痛点

- 官方文档术语密集
- 普通翻译缺少技术上下文
- 阅读时频繁切换工具会打断思路
## MVP 范围

- 右键菜单触发
- 侧边栏展示解释
- API Key 配置页
- OpenAI-compatible 调用
- 术语拆解
- 复制摘要
- 错误提示
## 非目标

- 账号系统
- 云端历史
- 团队共享
- 付费订阅
- 移动浏览器支持
## 核心功能

- 选中文本读取
- 右键菜单
- 侧边栏 UI
- 模型配置
- 解释结果渲染
- 复制按钮
## 技术约束

- API Key 只存浏览器本地
- 不自动抓取整页内容
- 限制单次解释文本长度
- 错误信息必须中文化
## 禁止事项

- 不要把 API Key 写入日志
- 不要上传未选中的页面内容
- 不要绕过网站安全限制

## Done When 验收标准

- 扩展可加载
- 选中文本可解释
- 错误提示不泄露 Key
- README 说明安装方式
