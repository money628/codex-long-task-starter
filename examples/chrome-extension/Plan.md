# Tech Explain Translator 开发计划

## 里程碑拆分

### 1. Extension MVP

目标：完成选中文本到侧边栏解释闭环

主要任务：
- 实现 manifest
- 实现右键菜单
- 实现配置页
- 实现侧边栏
- 接入模型调用

验收标准：
- 用户不离开网页即可看到解释
- 配置错误有中文提示
## 可能修改的文件

- 与 manifest 相关的页面、组件、状态和测试文件
- 与 background 相关的页面、组件、状态和测试文件
- 与 content-script 相关的页面、组件、状态和测试文件
- 与 side-panel 相关的页面、组件、状态和测试文件
- 与 options 相关的页面、组件、状态和测试文件
- 与 storage 相关的页面、组件、状态和测试文件
## 整体验收标准

- 右键菜单可触发
- 侧边栏可显示解释
- 配置页可保存 Key
- 复制摘要可用
- 无 Key 时提示配置
## 测试方式

- Chrome 开发者模式加载测试
- 模拟 API 响应
- 手动检查 storage 和控制台日志
## 失败修复策略

- 先复现错误并保留最小上下文
- 优先修复阻塞主流程的问题
- 每次修复后运行相关测试或构建
- 若需求不明确，将假设写入 Continuity.md 后继续推进
